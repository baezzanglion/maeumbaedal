# 📮 마음배달 (봉투)

익명 마니또를 뽑아, 각자에게 **개인 봉투 링크**를 만들어주는 작은 웹앱입니다.
운영자가 이름·미션·규칙만 넣고 **뽑기**를 누르면, 사람마다 다른 봉투 링크가 자동으로 생겨요. 서버 없이 링크 안에 결과가 담깁니다.

**라이브:** https://maeumbaedal.vercel.app

> 짝꿍 앱: [마니또 추리 대회](https://github.com/baezzanglion/maeumbaedal-quiz) — 배정을 맞혀보는 추리 대회

---

## 어떻게 작동하나요

- 운영자가 **브라우저에서** 참가자 이름 + 미션 + 규칙을 입력하고 **뽑기**를 누릅니다.
- 앱이 각 사람의 마니또를 계산해, 그 결과를 **링크(`?e=…`) 안에 base64로 담아** 사람 수만큼 봉투 링크를 만듭니다.
- 운영자는 `○○님의 봉투` 링크를 각자에게 1:1로 전달합니다.
- 받은 사람은 링크를 열어 봉투를 탭 → 자기 마니또와 이번 주 미션을 확인합니다.

> **서버·DB가 없습니다.** 뽑기와 배정은 전부 운영자 브라우저에서 일어나고 결과는 링크에만 존재해요.
> 그래서 여러 팀이 동시에 써도 서로 영향이 없고, 배포를 다시 해도 이미 보낸 링크는 그대로 살아있어요.

---

## 사용법

1. https://maeumbaedal.vercel.app 접속 (운영자)
2. 참가자 이름 입력 — 필요하면 **미션 10개 / 규칙 / 미션 제목 / 사람당 미션 개수**를 수정 (기본값이 채워져 있어요)
3. **뽑기** → 봉투 링크들이 생성됨
4. 각 `○○님의 봉투` 링크를 해당 사람에게 전달
5. 설정은 브라우저에 기억돼서(localStorage) 다음에 다시 열어도 유지돼요

---

## 다른 조가 복제해서 쓰는 법

이 앱은 **정적 파일 하나(`index.html`)**라서 배포가 아주 간단해요. 환경변수·DB 필요 없음.

```bash
git clone https://github.com/baezzanglion/maeumbaedal.git
cd maeumbaedal
npx vercel deploy --prod   # 또는 Netlify / GitHub Pages 등 아무 정적 호스팅
```

- 미션·규칙 기본값을 바꾸고 싶으면 `index.html` 안의 `DEFAULT_MISSIONS`, `DEFAULT_RULES`, `DEFAULT_MISSION_TITLE`을 수정하세요. (앱 첫 화면에서 그때그때 수정도 가능)
- 백엔드가 없어 GitHub Pages 같은 무료 정적 호스팅에도 그대로 올라가요.

## 기술 구조

- `index.html` — 앱 전체 (HTML + CSS + JS 한 파일). 브랜드 이미지는 base64로 인라인.
- 링크 포맷: `?e=<base64(JSON)>` — `{ t: 마니또, m: [미션들], r: [규칙들], mt: 미션제목 }`
- 구버전 호환: 예전 링크(`r`/`mt` 없는)도 기본 규칙/제목으로 정상 표시됩니다.

## 개인정보

배정 결과(누가 누구의 마니또인지)는 **서버에 저장되지 않습니다.** 각 봉투 링크 안에는 "받는 사람의 마니또"만 담겨 있고, 전체 배정표나 보낸 사람 정보는 들어있지 않아요.
